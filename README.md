# boast README

彩虹屁和一些小功能

## Features

- cmd+shift+k: 执行夸夸
- 去掉console选中代码，执行快捷键cmd+shift+r
- 右键查看文件详情
- 选中文本后右键 实现 rgb<=>hex互相转换
